# 👋 Hi, I'm Matan Adar

## 💫 About Me

I'm a **Mobile Application Developer** with a **B.Sc. in Computer Science**, currently developing production-ready cross-platform health-tech applications at **Yopi**.

I specialize in building mobile and wearable applications using **Flutter**, **SwiftUI**, **Firebase**, and **Bluetooth Low Energy (BLE)**. My experience includes developing real-time communication systems, integrating cloud services, implementing smartwatch connectivity, and designing scalable mobile solutions for iOS, Android, watchOS, and Wear OS.

I'm passionate about building high-quality software, solving complex engineering challenges, and continuously learning new technologies. I enjoy working across the full software development lifecycle—from architecture and implementation to optimization and deployment.
